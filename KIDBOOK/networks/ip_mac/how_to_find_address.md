# Как узнать свой IP и MAC-адрес

Каждое устройство в сети имеет адреса — IP и MAC. Иногда их нужно знать: при настройке сети, при решении проблем с интернетом или просто из любопытства. Подробнее о том, что это такое — в статье [IP и MAC-адреса](ip_and_mac.md).

---

## Публичный IP-адрес

Это адрес, который видит весь интернет — адрес твоего роутера. Самый простой способ узнать его — зайти на любой из этих сайтов:

- [whatismyip.com](https://whatismyip.com)
- [2ip.ru](https://2ip.ru)

Ты увидишь что-то вроде `85.249.112.47`. Это и есть твой публичный IP.

> Если ты подключён через мобильный интернет — это адрес твоего оператора. Если через домашний Wi-Fi — адрес роутера.

---

## Локальный IP-адрес

Это приватный адрес внутри твоей домашней сети — тот, который выдал роутер именно твоему устройству. Обычно начинается с `192.168.` или `10.`.

### Windows

Открой командную строку (Win + R → `cmd` → Enter) и введи:

```
ipconfig
```

Ищи строку **IPv4-адрес** — например:

```
IPv4-адрес. . . . . . . . . : 192.168.1.42
```

### macOS

Открой Терминал и введи:

```
ifconfig | grep "inet "
```

Или через графический интерфейс: Системные настройки → Сеть → выбери своё подключение.

### Linux

```
ip addr show
```

или

```
ifconfig
```

Ищи строку `inet` рядом с названием интерфейса (`eth0`, `wlan0` и т.п.).

---

## MAC-адрес

### Windows

В той же командной строке:

```
ipconfig /all
```

Ищи строку **Физический адрес** (Physical Address):

```
Физический адрес. . . . . . : A4-C3-F0-85-AC-2D
```

### macOS / Linux

```
ifconfig
```

Ищи строку **ether**:

```
ether a4:c3:f0:85:ac:2d
```

---

## ARP-кэш: кто ещё есть в сети

Чтобы увидеть все устройства, с которыми твой компьютер недавно общался в локальной сети (с их IP и MAC-адресами):

```
arp -a
```

Работает на Windows, macOS и Linux одинаково. Пример вывода:

```
192.168.1.1     d4-6e-0e-11-22-33   (роутер)
192.168.1.10    b8-27-eb-44-55-66
192.168.1.15    3c-22-fb-aa-bb-cc
```

## Шпаргалка

| Что узнать | Windows | macOS / Linux |
|---|---|---|
| Публичный IP | [whatismyip.com](https://whatismyip.com) | [whatismyip.com](https://whatismyip.com) |
| Локальный IP | `ipconfig` | `ifconfig` или `ip addr` |
| MAC-адрес | `ipconfig /all` | `ifconfig` (строка `ether`) |
| Устройства в сети | `arp -a` | `arp -a` |
