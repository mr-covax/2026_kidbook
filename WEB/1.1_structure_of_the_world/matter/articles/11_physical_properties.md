# Физические свойства материи

**Физические свойства** — характеристики [материи](01_matter.md), которые можно измерить или наблюдать без изменения химического состава вещества. В отличие от [химических свойств](12_chemical_properties.md), физические свойства не связаны с превращением одного вещества в другое.

## Масса

**Масса** — фундаментальное [физическое свойство](01_matter.md) материи, характеризующее её инертность (сопротивление ускорению) и гравитационное взаимодействие.

### Инертная масса

Определяется из второго закона Ньютона:

> F = m·a → m = F/a

Инертная масса показывает, насколько трудно изменить скорость тела.

### Гравитационная масса

Определяет интенсивность [гравитационного взаимодействия](14_fundamental_interactions.md):

> F = G·m₁·m₂/r²

Принцип эквивалентности Эйнштейна утверждает, что инертная и гравитационная массы равны — это один из фундаментов общей теории относительности и [уравнения E = mc²](18_mass_energy.md).

### Единицы измерения

Согласно [закону сохранения](15_conservation_law.md), суммарная масса-энергия замкнутой системы остаётся неизменной.

## Плотность

**Плотность** — масса вещества в единице объёма:

> ρ = m/V

### Плотность различных веществ

| Вещество | Состояние | Плотность, кг/м³ |
|----------|-----------|-------------------|
| Межзвёздная среда | [Плазма](08_plasma.md) | ~10⁻²¹ |
| Воздух (н.у.) | [Газ](07_gases.md) | 1,29 |
| Пробка | [Твёрдое](05_solids.md) | 120–240 |
| Вода | [Жидкость](06_liquids.md) | 1000 |
| Железо | Твёрдое | 7874 |
| Золото | Твёрдое | 19320 |
| Осмий | Твёрдое | 22590 |
| Белый карлик | Плазма | ~10⁹ |
| Нейтронная звезда | [Экзотическое](10_exotic_states.md) | ~10¹⁷ |

### Аномалия воды

Вода имеет максимальную плотность при **4 °C** (999,97 кг/м³), а при замерзании её плотность уменьшается (лёд плавает) — аномалия, обусловленная водородными [связями](04_molecular_level.md).

## Теплоёмкость

**Теплоёмкость** — количество теплоты, которое нужно сообщить телу для нагрева на 1 К.

### Удельная теплоёмкость

> c = Q / (m·ΔT)

Единица: Дж/(кг·К)

### Удельная теплоёмкость некоторых веществ

| Вещество | c, Дж/(кг·К) | Примечание |
|----------|--------------|------------|
| Вода | 4186 | Аномально высокая |
| Лёд | 2090 | |
| Алюминий | 897 | |
| Железо | 449 | |
| Медь | 385 | |
| Свинец | 129 | |
| Золото | 129 | |

### Молярная теплоёмкость

Для [идеальных газов](07_gases.md):
- При постоянном объёме: Cᵥ = (i/2)R, где i — число степеней свободы [молекулы](04_molecular_level.md)
- При постоянном давлении: Cₚ = Cᵥ + R

| Тип молекулы | i | Cᵥ | Cₚ |
|-------------|---|----|----|
| Одноатомная | 3 | (3/2)R | (5/2)R |
| Двухатомная | 5 | (5/2)R | (7/2)R |
| Многоатомная | 6 | 3R | 4R |

Для [твёрдых тел](05_solids.md) при высоких температурах действует закон Дюлонга-Пти: C ≈ 3R ≈ 25 Дж/(моль·К).

## Электропроводность

### Классификация материалов по электропроводности

| Класс | Удельное сопротивление, Ом·м | Примеры |
|-------|------------------------------|---------|
| Сверхпроводники | 0 (при T < Tс) | Nb₃Sn, YBa₂Cu₃O₇ |
| Проводники | 10⁻⁸ – 10⁻⁶ | Cu, Ag, Au, Al |
| Полупроводники | 10⁻⁶ – 10⁶ | Si, Ge, GaAs |
| Диэлектрики | 10⁶ – 10¹⁸ | Стекло, резина |

### Механизмы проводимости

- **Металлы**: свободные электроны (электронная проводимость). Связано с [металлической связью](04_molecular_level.md)
- **Электролиты**: подвижные ионы (ионная проводимость)
- **Полупроводники**: электроны и «дырки»
- **[Плазма](08_plasma.md)**: свободные электроны и ионы

### Зависимость от температуры

- Металлы: сопротивление **растёт** с температурой (R ∝ T)
- Полупроводники: сопротивление **уменьшается** с температурой
- Сверхпроводники: сопротивление падает до нуля ниже критической температуры — это [экзотическое состояние](10_exotic_states.md)
