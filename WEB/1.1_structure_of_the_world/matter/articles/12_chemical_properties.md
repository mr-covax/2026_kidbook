# Химические свойства материи

**Химические свойства** — способность [материи](01_matter.md) вступать в [химические реакции](https://ru.wikipedia.org/wiki/Химическая_реакция), в ходе которых одни вещества превращаются в другие с изменением состава и строения [молекул](04_molecular_level.md). В отличие от [физических свойств](11_physical_properties.md), химические свойства проявляются только при химическом взаимодействии вещества с другими веществами.

## Химические реакции

**Химическая реакция** — процесс превращения одних веществ (реагентов) в другие (продукты), сопровождающийся разрывом одних [химических связей](04_molecular_level.md) и образованием новых.

Признаки химических реакций:
- Изменение цвета
- Выделение или поглощение газа
- Образование осадка
- Выделение или поглощение теплоты
- Изменение запаха

При любой химической реакции выполняется [закон сохранения массы](15_conservation_law.md): суммарная масса реагентов равна суммарной массе продуктов.

## Типы химических реакций

### По числу и составу реагентов/продуктов

| Тип | Схема | Пример |
|-----|-------|--------|
| Соединение | A + B → AB | 2H₂ + O₂ → 2H₂O |
| Разложение | AB → A + B | 2H₂O → 2H₂ + O₂ |
| Замещение | AB + C → AC + B | Zn + CuSO₄ → ZnSO₄ + Cu |
| Обмен | AB + CD → AD + CB | NaCl + AgNO₃ → AgCl↓ + NaNO₃ |

### По тепловому эффекту

- **Экзотермические** (ΔH < 0) — с выделением теплоты (горение, нейтрализация)
- **Эндотермические** (ΔH > 0) — с поглощением теплоты (разложение, электролиз)

### По обратимости

- **Необратимые** — протекают до полного расходования реагентов
- **Обратимые** — протекают одновременно в прямом и обратном направлениях

### По наличию катализатора

- **Каталитические** — протекают с участием катализатора
- **Некаталитические** — без катализатора

## Энергетика реакций

### Энтальпия реакции

Тепловой эффект реакции при постоянном давлении:

> ΔH = ΣH(продукты) − ΣH(реагенты)

### Примеры энтальпий реакций

| Реакция | ΔH, кДж/моль |
|---------|--------------|
| Горение водорода: 2H₂ + O₂ → 2H₂O | −572 |
| Горение метана: CH₄ + 2O₂ → CO₂ + 2H₂O | −890 |
| Разложение воды: 2H₂O → 2H₂ + O₂ | +572 |
| Синтез аммиака: N₂ + 3H₂ → 2NH₃ | −92 |

В ядерных реакциях выделяемая энергия на несколько порядков больше и описывается [уравнением Эйнштейна](18_mass_energy.md) E = mc².

### Энергия Гиббса

Определяет самопроизвольность реакции:

> ΔG = ΔH − TΔS

Реакция самопроизвольна при ΔG < 0.

## Скорость химических реакций

Скорость реакции определяется изменением концентрации реагентов или продуктов в единицу времени.

### Факторы, влияющие на скорость

| Фактор | Влияние |
|--------|---------|
| Температура | Повышение T на 10 °C увеличивает скорость в 2–4 раза (правило Вант-Гоффа) |
| Концентрация | Увеличение концентрации ускоряет реакцию (закон действующих масс) |
| Катализатор | Снижает энергию активации |
| Площадь поверхности | Измельчение [твёрдых](05_solids.md) реагентов ускоряет реакцию |
| Давление | Влияет на реакции с участием [газов](07_gases.md) |

### Уравнение Аррениуса

> k = A · exp(−Eₐ / RT)

где k — константа скорости, A — предэкспоненциальный множитель, Eₐ — энергия активации.

## Химическое равновесие

В обратимых реакциях устанавливается **динамическое равновесие**, при котором скорости прямой и обратной реакций равны.

### Константа равновесия

Для реакции aA + bB ⇌ cC + dD:

> K = [C]ᶜ[D]ᵈ / [A]ᵃ[B]ᵇ

### Принцип Ле Шателье

При внешнем воздействии на систему в равновесии она смещается в направлении, противодействующем этому воздействию:

- Повышение температуры → смещение в сторону эндотермической реакции
- Увеличение давления → смещение в сторону меньшего числа молей [газа](07_gases.md)
- Увеличение концентрации реагента → смещение в сторону продуктов

## Основные химические свойства веществ

### Окислительно-восстановительные свойства

Связаны с передачей электронов между [атомами](03_atom_structure.md):
- **Окислители** — принимают электроны (F₂, O₂, KMnO₄)
- **Восстановители** — отдают электроны (Na, Zn, H₂)

### Кислотно-основные свойства

- **Кислоты** — доноры протонов H⁺ (HCl, H₂SO₄)
- **Основания** — акцепторы протонов (NaOH, Ca(OH)₂)
- **Амфотерные вещества** — проявляют и те, и другие свойства (Al₂O₃, ZnO)

### Электроотрицательность

Способность [атома](03_atom_structure.md) притягивать электронную плотность [связи](04_molecular_level.md). Определяет тип химической связи и реакционную способность.

| Элемент | Электроотрицательность (шкала Полинга) |
|---------|---------------------------------------|
| F | 4,0 (максимальная) |
| O | 3,5 |
| N | 3,0 |
| C | 2,5 |
| H | 2,1 |
| Na | 0,9 |
| Cs | 0,7 (минимальная) |

Химические реакции на ядерном уровне (ядерные реакции) могут приводить к превращению одних [элементарных частиц](13_elementary_particles.md) в другие и даже к рождению [антиматерии](16_antimatter.md).
